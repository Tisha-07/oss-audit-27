## 📜 Project Overview
This project is part of the Open Source Software course. It involves:

- Studying the philosophy and principles of open source

- Exploring its role in the Linux ecosystem
- Writing shell scripts to demonstrate Linux and automation skills

---

## 💻 Shell Scripts Description

### 🔹 Script 1: System Identity Report
- Displays system details like:
  - Kernel version
  - Logged-in user
  - System uptime
  - Date and time
- Demonstrates use of variables and system commands

---

### 🔹 Script 2: FOSS Package Inspector
- Checks if a package is installed
- Displays version, license, and summary
- Uses `if-else`, `case`, and package management commands

---

### 🔹 Script 3: Disk and Permission Auditor
- Analyzes important system directories
- Displays:
  - Directory size
  - Permissions
  - Owner details
- Uses loops and text processing tools

---

### 🔹 Script 4: Log File Analyzer
- Reads a log file line-by-line
- Counts occurrences of a keyword (default: "error")
- Displays summary of results
- Uses `while-read`, conditionals, and counters

---

### 🔹 Script 5: Open Source Manifesto Generator
- Takes user input:
  - Tool used daily
  - Meaning of freedom
  - What user will build
- Generates a personalized manifesto
- Saves output to a `.txt` file

---

## ⚙️ How to Run the Scripts

### Step 1: Give execute permission
```bash
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
```

### Step 2: Run the scripts
```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog
./script5.sh
```
